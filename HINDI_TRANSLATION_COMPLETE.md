## ЁЯЪА How to Extend

### Add Translation to Any Component:

```tsx
import { useLanguage } from "@/contexts/LanguageContext";

export function MyComponent() {
  const { t, language } = useLanguage();
  
  return (
    <div>
      <h1>{t('section.key')}</h1>
      <p>Current language: {language}</p>
    </div>
  );
}
```

### Add New Translation Key:

1. Add to `en.json`:
   ```json
   { "mySection": { "myKey": "English text" } }
   ```

2. Add to `hi.json`:
   ```json
   { "mySection": { "myKey": "рд╣рд┐рдВрджреА рдкрд╛рда" } }
   ```

3. Use in component:
   ```tsx
   const text = t('mySection.myKey');
   ```
